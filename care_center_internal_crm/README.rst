========================
Care Center Internal CRM
========================

Handle internal calls between employees working on Tasks.

Usage
=====

Install module and record phone call. Click the "Make Internal Call" button on Tasks.

Bug Tracker
===========

Bugs are tracked on `GitHub Issues <https://github.com/thinkwelltwd/care_center/issues>`_.
In case of trouble, please check there if your issue has already been reported.
